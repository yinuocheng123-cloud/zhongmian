# 一、说明目的

本说明用于在中眠网上线后，通过一组固定页面截图，快速确认前台公开访问、后台登录管理、内容展示与 AI 编辑部链路是否已经正常工作，从而帮助客户判断“站点已成功上线并可正常使用”。

# 二、截图顺序说明

建议截图顺序按“前台公开 → 详情页 → 后台登录 → 后台管理 → AI 编辑部”来走。  
每张截图都建议保留浏览器地址栏，方便确认页面路径正确。

# 三、详细截图检查项

**1. 首页**
- 页面：`/`
- 要截图什么：
  - 首页首屏
  - 顶部导航
  - 核心入口区
  - 页面主标题
- 正确结果：
  - 页面可正常打开
  - 首页不是空白页
  - 能看出中眠网是“睡眠知识入口 + 品牌入口 + 行业参考入口”
  - 没有报错、错版、明显占位文案

**2. 知识列表页**
- 页面：`/knowledge`
- 要截图什么：
  - 页面首屏
  - 至少 1 组知识卡片
  - 如有筛选条件，一并截上
- 正确结果：
  - 列表不为空
  - 有真实知识内容
  - 页面更像知识入口 / 行业趋势入口，不像普通新闻流

**3. 词库列表页**
- 页面：`/terms`
- 要截图什么：

  - 至少 1 组词条卡片
- 正确结果：
  - 列表不为空
  - 能看到词条名和定义
  - 页面更像术语目录 / 概念入口，不是文章列表

**4. 品牌列表页**

  - 可看到品牌名、简介、地区或主营方向等结构信息

客户看到截图后，至少应能确认：

1. 前台首页和三条主线都能打开  
2. 前台不是空站，已有真实内容  
3. 三类详情页都能访问  
4. 后台可登录、可管理内容  
5. AI 编辑部已接入内容生产流程  
6. 系统不是 demo，而是可运营系统
