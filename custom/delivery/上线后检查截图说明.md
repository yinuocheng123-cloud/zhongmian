# 一、说明目的

本说明用于在中眠网上线后，通过一组固定截图快速确认：前台主入口可访问、扩展栏目已落地、后台可登录并可运营、AI 编辑部链路正常，从而判断本次上线是否成功。

# 二、截图顺序说明

建议截图顺序按“前台公开 → 扩展栏目 → 详情页 → 后台登录 → 后台管理 → AI 编辑部”来走。  
每张截图都建议保留浏览器地址栏，方便确认页面路径正确。
